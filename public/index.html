✅ API is running

Welcome to the Clinic Management System Backend API

📡 Available Endpoints:

🔐 Authentication

POST /api/auth/login - User login
POST /api/auth/logout - User logout (Auth required)
GET /api/auth/me - Get current user (Auth required)
POST /api/auth/refresh - Refresh token (Auth required)

👥 User Management

GET /api/user - List all users
POST /api/user - Create new user
PATCH /api/user/{id}/password - Change password

👨‍⚕️ Staff Management

GET /api/staff - List all staff
POST /api/staff - Create new staff

🏥 Patient Management

GET /api/pasien - List all patients
POST /api/pasien - Create new patient

📋 Queue Management

GET /api/antrian - List all queues
POST /api/antrian - Create new queue

📅 Appointment Management

GET /api/appointment - List all appointments
POST /api/appointment - Create new appointment
PATCH /api/appointment/{id}/status - Update status

💊 Treatment Management

GET /api/treatment - List all treatments
POST /api/treatment - Create new treatment

📦 Inventory Management

GET /api/barang - List all items
POST /api/barang - Create new item
PATCH /api/barang/{id}/stock - Update stock

🏢 Supplier Management

GET /api/supplier - List all suppliers
POST /api/supplier - Create new supplier

🔄 Item Conversion

GET /api/konversi-stok - List all conversions
POST /api/konversi-stok - Create new conversion

🏭 Production Management

GET /api/produksi-barang - List all productions
POST /api/produksi-barang - Create new production

📊 Stock Opname

GET /api/stok-opname - List all stock opnames
POST /api/stok-opname - Create new stock opname

📋 Stock Card

GET /api/kartu-stok - List all stock cards
POST /api/kartu-stok - Create new stock card

🎉 Promotions

GET /api/promo - List all promotions
POST /api/promo - Create new promotion
POST /api/promo/validate - Validate promotion

👑 Membership

GET /api/membership - List all memberships
POST /api/membership - Create new membership
POST /api/membership/{id}/add-points - Add points
POST /api/membership/{id}/use-points - Use points

🏢 Branch Management

GET /api/cabang - List all branches
POST /api/cabang - Create new branch

🏦 Bank Management

GET /api/bank - List all banks
POST /api/bank - Create new bank

🏢 Company Management

GET /api/perusahaan - List all companies
POST /api/perusahaan - Create new company

💚 Health Check

GET /api/health - System status
